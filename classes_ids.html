Classes

O ponto (.) define uma classe. É como se fosse uma variável no python.

Por exemplo, usei uma classe 'roxo'{} para criar uma categoria de cor roxa.
Assim, posso caracterizar qualquer coisa apenas colocando class = "roxo."

Lembrando que a definição da classe é escrita dentro da tag style que por
sua vez está dentro da tag head .

Exemplo

A ideia é usar um seletor para categorizar coisas gerais e usar classes
para formatações especifícas.

IDs

O ID pode parecer com a classe. Mas o ID é utilizado apenas uma única vez
em cada página e assim como a classe, é definido na tag style começando com (#).
Como exemplo usei # principal (sem espaço) para definir o background-color marrom
na tag body .
